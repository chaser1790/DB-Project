# Проект "Вакансии на HeadHunter"

Этот проект позволяет создавать базу данных о вакансиях определенных компаний, размещенных на веб-сайте HeadHunter, и проводить с ними некоторые общие аналитические операции.

Проект включает в себя скрипты на языке Python, SQL-запросы и работает с базой данных PostgreSQL. База данных будет заполнена данными, получаемыми от API веб-сайта `HeadHunter`.

## Установка и запуск проекта

### Установка и активация виртуального окружения

Во-первых, проверьте, установлен ли интерпретатор Python версии не ниже 3.6.

Этот шаг позволяет изолировать зависимости данного проекта от любых других, установленных на вашем компьютере. В зависимости от вашей операционной системы и инструментов, выберите подходящий способ установки и активации виртуального окружения:

#### Для MacOS и терминала:

Создание и активация виртуального окружения:

```bash
python3 -m venv env
source env/bin/activate
```

#### Для Windows и терминала:

Создание и активация виртуального окружения (с использованием PowerShell):

```bash
py -3 -m venv env
.\env\Scripts\activate
```

#### Для PyCharm через терминал:

В PyCharm включите terminal (Alt+F12 в Windows/Linux, ⌥F12 в MacOS). Затем выполните следующие команды:

Windows:

```bash
cd path_to_project
py -3 -m venv env
.\env\Scripts\activate
```

MacOS:

```bash
cd path_to_project
python3 -m venv env
source env/bin/activate
```

`path_to_project` - это путь к директории вашего проекта.

### Установка необходимых зависимостей

Устанавливаем необходимые зависимости:

```bash
pip install -r requirements.txt
```

### Создание и заполнение базы данных

Создайте PostgreSQL базу данных и пользователя со всеми необходимыми правами на взаимодействие с ней.

В корне проекта создайте файл `database.ini` с информацией для подключения к созданной базе данных. Образец файла `database.ini` находится в конце руководства!


Запустите файл `create_db_insert_tables.py` для создания таблиц в базе данных и заполнения их данными через API веб-сайта HeadHunter.


### Запуск основного скрипта

Запустите основной скрипт проекта `main.py`.

```bash
python main.py
```

## Работа с проектом 

После запуска `main.py` вы увидите консольное меню со следующими пунктами:

1. Отобразить список всех компаний и количество вакансий у каждой компании.
2. Отобразить список всех вакансий.
3. Рассчитать среднюю зарплату по вакансиям.
4. Отобразить список вакансий с зарплатой выше средней.
5. Поиск вакансий по ключевому слову.
0. Выйти из программы.

## Примеры файлов для работы программы

### Requirements.txt

Пример содержимого файла `requirements.txt`:

```
requests~=2.31.0
psycopg2~=2.9.9
```

### Database.ini

Пример содержимого файла `database.ini`:

```ini
[postgresql]
host=Указать название хоста (например localhost)
dbname=(ПОЛЕ ОСТАВИТЬ ПУСТЫМ!!!)
user=Ваше имя пользователя
password=Ваш пароль
```

Вводите номер выбранного действия в консоли и следуйте дальнейшим указаниям. 

Когда вы закончите работу с проектом, выберите пункт "0.Выйти из программы" для корректного завершения работы.
